# 💬 Keryx (Family Link)

## The Problem 🤦‍♂️
Mom and Brother need to communicate. 
- Mom has one phone.
- Brother has a Samsung with Knox restrictions that hates background apps more than a cat hates water. 
- They both use the SAME WhatsApp number across devices (don't ask why). 
- Knox aggressively kills background apps unless explicitly allowed. No WhatsApp calls are possible because of the shared number and Knox being an absolute dictator.
- They just need to talk for like 5 minutes for emergency messages. We need a communication channel that sleeps quietly in the background (using almost zero battery and RAM) and only wakes up when Mom hits the "Call me" button.

## The Solution 🦸‍♂️
Enter **Keryx**. 
- A lightweight app that doesn't keep a permanent, battery-draining voice connection open.
- When they want to talk, Mom sends a "Call me" text.
- Brother gets a lightweight background notification that slips right past Knox's sleepy guards.
- Brother opens Keryx, they connect directly via WebRTC, they talk, they hang up.
- The app goes back to sleep, saving battery and RAM. Everyone is happy. Take that, Samsung Knox!
